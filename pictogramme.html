Pictogrammes de Sécurité en Biotechnologie

Corrosion

Description : Pour les substances qui provoquent des brûlures graves et des dommages aux tissus vivants, aux métaux et à d'autres matériaux.

Exemple : Acides, bases fortes (ex : acide sulfurique, hydroxyde de sodium).

Flamme

Description : Pour les substances qui s'enflamment facilement en présence de chaleur, de flamme ou de sources d'étincelles.

Exemple : Solvants organiques, éthanol, éther.

Flamme sur un cercle

Description : Pour les gaz inflammables, les substances auto-inflammables ou celles qui peuvent produire un gaz inflammable au contact de l'air.

Exemple : Hydrogène, produits chimiques qui réagissent spontanément avec l'air ou l'humidité.

Explosif

Description : Pour les substances qui peuvent exploser sous l'effet de chaleur, de choc ou de friction.

Exemple : TNT, peroxyde d'hydrogène concentré.

Gaz sous pression

Description : Pour les gaz comprimés, liquéfiés, dissous ou réfrigérés qui peuvent causer des blessures en cas de fuite ou de rupture du conteneur.

Exemple : Gaz de laboratoire comme l'azote liquide, l'hélium, le dioxyde de carbone.

Santé (Toxic, Health Hazard)

Description : Pour les substances qui peuvent causer des effets graves pour la santé comme des maladies, des lésions des organes, la reproduction, ou des effets carcinogènes.

Exemple : Produits chimiques cancérogènes, agents toxiques.
