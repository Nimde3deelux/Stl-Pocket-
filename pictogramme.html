Pictogrammes de Sécurité en Biotechnologie

Corrosion

Description : Pour les substances qui provoquent des brûlures ou des dommages importants aux tissus.

Exemple : Acides, bases fortes (ex : acide sulfurique, hydroxyde de sodium).

Flamme

Description : Pour les substances inflammables ou auto-inflammables.

Exemple : Solvants organiques, éthanol, éther.

Flamme sur un cercle

Description : Pour les gaz inflammables et les produits qui produisent des gaz inflammables.

Exemple : Hydrogène, produits chimiques réagissant avec l'air.

Explosif

Description : Pour les substances explosives.

Exemple : TNT, peroxyde d'hydrogène concentré.

Gaz sous pression

Description : Pour les gaz comprimés, liquéfiés ou dissous.

Exemple : Azote liquide, hélium, dioxyde de carbone.

Santé (Toxic, Health Hazard)

Description : Pour les substances nocives, toxiques, ou cancérogènes.

Exemple : Produits chimiques cancérogènes, agents toxiques.

Santé (Exclamation)

Description : Pour les dangers pour la santé moins graves.

Exemple : Irritants légers, substances allergènes.

Santé (Point d'interrogation)

Description : Pour les substances dont les effets sont inconnus.

Exemple : Substances avec des risques peu documentés.

Danger pour l'environnement

Description : Pour les produits nuisibles à l'environnement, en particulier aquatique.

Exemple : Produits chimiques toxiques pour la vie aquatique.

Infectieux (Biohazard)

Description : Pour les agents biologiques pouvant causer des infections.

Exemple : Agents pathogènes, bactéries pathogènes.

Cellules et bactéries

Description : Pour indiquer les risques biologiques spécifiques.

Exemple : Cultures cellulaires, bactéries pathogènes.

Sécurité (Casque de protection)

Description : Pour le port obligatoire de protections comme des lunettes ou un casque.

Exemple : Équipements de sécurité sur site.

Gants

Description : Pour le port de gants lors de la manipulation de produits dangereux.

Exemple : Gants en nitrile ou latex.

Masque de protection

Description : Pour le port obligatoire d'un masque ou d'un respirateur.

Exemple : Masques N95, masques chirurgicaux.

Chaussures de sécurité

Description : Pour l'obligation de porter des chaussures de sécurité.

Exemple : Chaussures renforcées pour protection des pieds.

Vêtements de protection

Description : Pour porter des blouses ou vêtements de laboratoire.

Exemple : Blouses de laboratoire, combinaisons de protection.

Premiers secours

Description : Pour les produits nécessitant une assistance rapide en cas d'accident.

Exemple : Kits de premiers secours, solutions de lavage oculaire.
